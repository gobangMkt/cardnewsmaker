# 카드뉴스 이모지 사용 가이드

## 1. TossFace 폰트 적용

Puppeteer 환경에서 이모지가 시스템마다 다르게 렌더링되는 문제를 방지하려면 TossFace 폰트를 명시적으로 로드해야 합니다.

### HTML에 추가
```html
<link rel="stylesheet" href="https://static.toss.im/tossface/cdn/tossface.css">
```

### CSS 적용
```css
.hl-icon, .emoji {
  font-family: 'TossFaceFontMac', 'Apple Color Emoji', 'Segoe UI Emoji', sans-serif;
}
```

> **주의**: TossFace CDN이 외부 요청이므로 Puppeteer에서 `waitUntil: 'networkidle0'` 옵션 필요.

## 2. 카드뉴스 내 이모지 위치 규칙

| 위치 | 클래스 | 용도 | 권장 크기 |
|-----|--------|-----|---------|
| 헤드라인 앞 아이콘 | `.hl-icon` | 슬라이드 주제 상징 | `font-size: 50px` |
| 본문 강조 포인트 | 인라인 텍스트 | 단락 내 강조 | `font-size: 36px` |
| 커버 타이틀 | — | **사용 금지** (배경 사진과 충돌) | — |

**1슬라이드 = 이모지 최대 1개** (헤드라인 아이콘만)

## 4. HTML 사용 예시

### list 타입 헤드라인
```html
<!-- 이슈 분석 슬라이드 -->
<div class="headline">
  <div class="hl-icon">🔥</div>
  <div class="hl-text">지금</div>
  <div class="hl-badge">급등 지역</div>
  <div class="hl-text">TOP 5</div>
</div>

<!-- 팁/가이드 슬라이드 -->
<div class="headline">
  <div class="hl-icon">💡</div>
  <div class="hl-text">청약</div>
  <div class="hl-badge">당첨 확률</div>
  <div class="hl-text">높이는 법</div>
</div>

<!-- 주의/리스크 슬라이드 -->
<div class="headline">
  <div class="hl-icon">⚠️</div>
  <div class="hl-text">이것만은</div>
  <div class="hl-badge">조심</div>
  <div class="hl-text">하세요</div>
</div>
```

### 커버 타이틀 (이모지 없이)
```html
<!-- 커버에는 이모지 쓰지 않음 — 배경 이미지와 충돌 -->
<div class="headline">
  전셋값 반등<br><span class="key">진짜</span> 시작됐나
</div>
```

## 5. AI 생성 시 이모지 선택 규칙

카드뉴스 자동 생성 시 GPT/Claude에게 이모지를 고르게 할 때 적용할 프롬프트 규칙:

```
슬라이드 헤드라인 앞 이모지를 1개 선택하세요.
- 카테고리: 부동산/금융/뉴스강조/인사이트/감정 중 하나
- 커버 슬라이드는 이모지 없음
- 반드시 단일 유니코드 이모지 1개만 출력
- 슬라이드 내용과 직관적으로 연결되어야 함
- 추천: 📌 🔥 💡 📊 ⚠️ ✅ 🎯 🔑 📈
```
